# Prompt-injection-detection
Overview

AI Prompt Injection Detector is a cybersecurity-focused application designed to identify and analyze prompt injection, jailbreak, and instruction override attacks targeting Large Language Models (LLMs). The system combines semantic AI analysis, regex-based detection, OCR image scanning, and weighted risk scoring to detect both direct and indirect prompt manipulation attempts in real time.

The project demonstrates how AI-driven security systems can be used to improve the safety and reliability of modern conversational AI systems against adversarial inputs and social engineering attacks.

Features
Real-time prompt injection detection
Semantic analysis using fine-tuned LLMs
Regex-based keyword detection
OCR-based image prompt scanning
Weighted risk scoring system
Detection of jailbreak and policy bypass attempts
Detection of hidden prompt extraction attempts
User-friendly Streamlit interface
Cloudflare tunneling support for remote deployment
